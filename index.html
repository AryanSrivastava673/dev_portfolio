QR / NFC Restaurant Ordering SaaS

nairamenus.in ↗

Led development of an NFC-powered digital menu and ordering platform built to replace paper menus, now live and onboarding real cafe clients in a paid pilot. Drove architecture, backend, frontend, and deployment across the full stack.

Multi-Tenant Architecture

Multi-tenant, multi-outlet data isolation with RBAC (Owner/Manager/Admin) and cookie-token outlet switching under a single Spring Boot backend.

JWT Auth System

Stateless JWT via HttpOnly cookies with a 3-tier token resolution chain, support impersonation for debugging, and a custom rate-limiting filter.

Real-Time Ordering

Kitchen and table-call updates over WebSocket (STOMP/SockJS) with topic-scoped subscriptions — sub-second order-to-kitchen latency.

Thermal Printing

Driverless thermal receipt printing via Web Serial API (ESC/POS, 58/80mm), eliminating driver installs, with automatic PDF fallback.

Infrastructure

4 services on Docker Compose + Nginx + Cloudflare Tunnel on an Azure VM, with GitHub Actions CI/CD and Azure Database for MySQL.

Integrations & Testing

UrbanPiper webhook ingestion, Meta WhatsApp Cloud API notifications, and 31 JUnit test suites covering auth, billing, and ordering.
